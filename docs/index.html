Pharos Security Suite

Four Modules

Full-surface security coverage

Each module is independent — use one, or compose all four into a unified security pipeline for autonomous agents.

01

Skill Inspector v1.0 Python

Detect prompt injection, data leakage, vulnerable dependencies, and on-chain risks in AI agent skills before you install or publish them.

  • Zero runtime dependencies (Python 3.10+)
  • Multi-line taint tracking: key → sink
  • Live OSV.dev CVE lookups + offline fallback
  • Output: terminal, JSON, Markdown, SARIF
Full docs →
02

Contract Inspector v1.1 Node.js

Point it at any contract on Pharos and get function inventory, proxy resolution, interface detection, and risk scoring — straight from bytecode.

  • ABI-free: works on unverified contracts
  • Proxy detection: EIP-1167/1967, UUPS, OZ
  • Fingerprint: ERC-20/721/1155, Ownable, Pausable
  • CLI + HTTP API for agent integration
Full docs →
03

Address Intelligence v1.0 Node.js

EOA vs contract detection, native + ERC-20 holdings, behavioral classification, and a deterministic 0–100 risk score — from on-chain data.

  • RPC-only mode works when explorer is down
  • Classification: Whale/Bot/MEV/Dormant, etc.
  • Risk levels: LOW → CRITICAL
  • Graceful degradation with confidence flags
Full docs →
04

Agent Trust Layer planned

Cryptographic attestation engine that seals audit results from modules 1–3 with verifiable signatures — agent-to-agent trust without a central authority.

  • Signed attestations (SHA-256 + Ed25519)
  • On-chain attestation registry on Pharos
  • Cross-module unified trust score
  • Verification SDK for consuming agents

Quick Start

Running in under a minute

Clone the suite and scan a live mainnet contract or address right away.

# Clone the suite
git clone https://github.com/arraya20/pharos-security-suite.git
cd pharos-security-suite

# Module 2 — inspect a live mainnet contract
cd contract-inspector && npm install
node inspect.js 0x000000000022D473030F116dDEE9F6B43aC78BA3 --network mainnet

# Module 3 — profile any address
cd ../address-intelligence && npm install
node scripts/inspect.mjs 0x126cC4E8f6c24fdBe65e07AA8CaDB6dB1ec655e2 --network mainnet
